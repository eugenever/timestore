# Timestore

Серверное приложение, которое собирает измерения с разных типов устройств/радиотехнологий и сохраняет их в формате `lance` совместимом с `Apache Arrow`. В последующем с помощью указанных измерений выполняется локализация устройств на основе мультилатерации (multilateration).

Приложение может принимать данные по `http` (в качестве сервера используется `Axum`) либо через `NATS`.

Для передачи больших объемов данных используется потоковый формат `Apache IPC`.
